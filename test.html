즐거운 정보~ 가로세로 낱말 퍼즐-1

가로퍼즐 Hint

  • 1. 컴퓨터 언어를 사용하여 프로그램을 작성하는 행위.
  • 3. 어떤 문제를 해결하기 위한 단계적이고 논리적인 절차나 방법들의 집합.
  • 4. 컴퓨터 시스템을 작동시키고 특정 작업을 수행하도록 하는 프로그램과 데이터.
  • 8. 컴퓨터가 처리한 결과를 모니터나 프린터 등으로 외부에 보여주는 과정.
  • 6. 컴퓨터 시스템에 침투하여 정보를 손상시키거나 오작동을 유발하는 악성 프로그램.
  • 10. 8개의 비트(bit)가 모여 하나의 문자를 표현하는 정보의 단위.

세로퍼즐 Hint

  • 2. 컴퓨터 시스템이 작업을 수행하는 데 필요한 모든 유무형의 자원을 총칭하는 개념.
  • 5. 기존의 데이터 처리 방식으로는 수집, 저장, 분석하기 어려울 정도로 방대하고 다양한 형태의 대규모 데이터를 의미함.
  • 7. 컴퓨터를 구성하는 물리적인 부품이나 장치들.(예: CPU, 메모리, 키보드)
  • 9. 키보드나 마우스 등을 통해 데이터를 컴퓨터로 받아들이는 과정.
  • 11. 정보의 가장 작은 단위로, 0 또는 1의 두 가지 값 중 하나를 가짐.
